## 功能
- ✅ 实时温度显示
- ✅ 支持万物互联MQTT
- ✅ 接入网络API，显示励志英语
- ✅ OTA远程升级


# 硬件配置

1.ESP8266。

2.AHT10。

3.SH1106 1.3寸OLED屏幕。

4.涡扇。

5.散热片*2。

6.NVME2230固态硬盘。
